# Marketing Funnel A/B Testing Analysis

This project analyzes marketing funnel performance using A/B testing and statistical hypothesis testing.

## Tools Used

- Python
- Pandas
- Matplotlib
- MLflow
- Statistical Z-Test

## Project Workflow

1. Data Cleaning
2. Conversion Rate Analysis
3. A/B Testing using Z-Test
4. Funnel Visualization
5. Experiment Tracking with MLflow

## Key Result

Control Conversion Rate: 12.03%  
Treatment Conversion Rate: 11.88%

P-value = 0.189

Result: No statistically significant difference between landing pages.
